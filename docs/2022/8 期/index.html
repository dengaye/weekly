ECMAScript​

ECMAScript 2022 新特性​

  • Top-level Await
  • Object.hasOwn()
  • atArray.prototype.at()
  • error.cause
  • 正则表达式匹配索引
  • 类的实例成员

ECMAScript 2021 新特性​

  • String.prototype.replaceAll
  • Promise.any()
  • 新增逻辑赋值操作符： ??=、&&=、 ||=
  • WeakRefs
  • 下划线 (_) 分隔符

ECMAScript 2020 新特性​

  • Optional Chaining 可选链式调用
  • Nullish Coalescing 空值合并
  • Private Fields 私有字段
  • Static Fields 静态字段
  • Promise.allSettled 方法
  • Dynamic Import 动态引入
  • String.prototype.matchAll()
  • BigInt：内置对象
  • globalThis：提供了一个标准的方式来获取不同环境下的全局 this 对象（也就是全局对象自身）
